# Personal Portfolio Website

## Project Description
A responsive personal portfolio website showcasing skills, projects, and contact information.

## Features
- Responsive design
- Semantic HTML5
- CSS Grid & Flexbox
- Hover effects
- Contact form

## Technologies Used
HTML5, CSS3, JavaScript, Git

## Setup Instructions
1. Clone repository
2. Open index.html using Live Server

## Testing
Tested on Chrome and mobile view
